How should the single page be structured?

Currently Home has: Hero + Key Numbers + About preview. The full About lives on a separate page. What do we do on the merged single page?

A — Merge preview into full About

One continuous About section. Remove the "Read More →" link. The full summary + sector tags replace the preview.

Page structure
🟦 Hero
🟦 Key Numbers
🟦 About full summary + sectors
🟦 Experience
🟦 Skills
🟦 Projects
🟦 Contact

Pros

  • Clean, no duplication
  • Simpler to implement

Cons

  • Longer scroll before Experience

B — Keep preview, add full About below

Hero still has a short About preview. "Read More →" scrolls down to a full About section further down. Two About blocks.

Page structure
🟦 Hero
🟦 Key Numbers
🟦 About preview short
🟦 About full nav target
🟦 Experience
🟦 Skills
🟦 Projects
🟦 Contact

Pros

  • Preserves existing Home feel

Cons

  • Repeated About content
  • More complex

My recommendation: A — one clean About section, no duplication.
